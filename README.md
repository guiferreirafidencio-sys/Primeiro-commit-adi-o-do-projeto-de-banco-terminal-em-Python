# 💰 Sistema Bancário em Python

Projeto simples de um sistema bancário feito em Python no terminal.

## 📌 Funcionalidades

- Criar conta
- Login de usuário
- Saldo inicial
- Transferência entre usuários
- Histórico de transações
- Visualizar saldo
- Interface simples no terminal

## 🛠 Tecnologias usadas

- Python
- JSON (armazenamento de dados)
- datetime
- os

## 📂 Estrutura do projeto

arquivo.py → sistema principal  
cadastro.py → funções de cadastro e login  
dados.json → armazenamento dos usuários

## 🎯 Objetivo

Praticar lógica de programação, manipulação de arquivos JSON e criação de sistemas simples no terminal.

## ▶ Como executar

1. Instale o Python
2. Baixe o projeto
3. Execute:

```bash
python arquivo.py
