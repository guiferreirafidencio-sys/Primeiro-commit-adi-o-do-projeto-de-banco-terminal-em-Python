# Primeiro-commit-adi-o-do-projeto-de-banco-terminal-em-Python
Sistema bancário em Python executado no terminal. Permite criar contas, fazer login, transferir dinheiro entre usuários, consultar saldo e visualizar histórico de transações. Os dados são armazenados em um arquivo JSON. Projeto feito para praticar lógica, funções e manipulação de arquivos.
